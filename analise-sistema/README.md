##Descrição da Atividade
Atividade tem como objetivo aplicar conceitos de Engenharia Reversa, Arquitetura de Software, Design, Modelagem UML e Padrões de Projeto a partir da análise de um sistema real em produção.

O sistema analisado foi:  
https://tropykalypizzaselanches.com.br/

A análise foi realizada sem acesso ao código-fonte, utilizando apenas a observação do comportamento do sistema.

Objetivos

-Compreender a estrutura de um sistema real  
-Aplicar conceitos de arquitetura e design de software  
-Identificar padrões de projeto  
-Desenvolver modelagem UML baseada em inferência  
-Comparar sistema real com sistema didático
