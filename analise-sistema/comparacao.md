## Parte 5 – Comparação com Sistema Didático

| Critério        | Sistema Real                                                                 | Sistema Didático                                                             |
|----------------|-----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| Arquitetura     | Arquitetura web cliente-servidor, com indícios de separação em camadas      | Arquitetura simples, geralmente sem divisão clara de camadas                |
| Coesão          | Coesão moderada a alta, com funções bem definidas (produtos, carrinho etc.) | Baixa coesão, com responsabilidades misturadas                              |
| Acoplamento     | Acoplamento moderado, podendo haver dependência entre interface e lógica    | Alto acoplamento, dificultando manutenção                                   |
| Organização     | Estrutura mais organizada, voltada para uso real                            | Organização simples, focada apenas no aprendizado                           |
| Flexibilidade   | Maior flexibilidade, permitindo expansão e melhorias                        | Baixa flexibilidade, com dificuldade para evoluir o sistema                 |

### Explicação

O sistema real apresenta uma estrutura mais organizada e próxima de padrões profissionais, com melhor separação de responsabilidades e maior preocupação com a experiência do usuário. Já o sistema didático é simplificado e voltado apenas para fins de aprendizado, o que resulta em menor organização, maior acoplamento e menor flexibilidade.
