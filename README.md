# Programação Web I - Unidade 1

Repositório contendo as atividades da disciplina de Programação Web I

## Estrutura do Projeto

unidade-1/
- atividade-1-classificacao/
- atividade-2-formulario/
- atividade-3-login/
- atividade-4-cookies/
- atividade-5-desafio-integrador/

## Descrição das Atividades

### Atividade 1 - Classificação Acadêmica
Sistema que recebe uma nota e informa a situação do aluno (Aprovado, Recuperação ou Reprovado).

### Atividade 2 - Formulário
Formulário de cadastro com validação de campos utilizando método POST.

### Atividade 3 - Login
Sistema de login com:
- autenticação de usuário
- uso de sessão
- proteção de página
- logout

### Atividade 4 - Cookies
Sistema que armazena o nome do usuário por 7 dias utilizando cookies.

### Atividade 5 - Desafio Integrador
Sistema completo com:
- login com sessão
- cadastro via formulário
- armazenamento em array
- exibição dos dados cadastrados
- validação com função

## Tecnologias utilizadas
- PHP
- HTML
